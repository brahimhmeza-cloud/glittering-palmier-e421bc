## الخطوة 2: نشر الموقع على Vercel (مجاني)

### 2.1 إنشاء حساب Vercel
1. اذهب إلى **https://vercel.com**
2. اضغط **Sign Up**
3. اختر **Continue with GitHub** (أو Google)
4. أكمل التسجيل

### 2.2 رفع المشروع
1. اضغط **Add New Project**
2. اختر **Upload** (ارفع مجلد `dist`)
3. انتظر حتى يكتمل الرفع
4. سيظهر لك رابط مثل: `bh-cosmetics.vercel.app`

### 2.3 ربط النطاق
1. من لوحة تحكم المشروع، اختر **Settings**
2. اضغط **Domains**
3. أدخل النطاق الخاص بك (مثلاً: `bhcosmetics.ma`)
4. اضغط **Add**

---

## الخطوة 3: إعداد DNS Records

### في موقع النطاق (مثلاً Genious.ma):
1. اذهب إلى **إدارة النطاقات** → **DNS Management**
2. أضف السجلات التالية:

```
Type: A
Name: @
Value: 76.76.21.21
TTL: 3600
```

```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: 3600
```

### مثال بصري:
```
┌─────────────────────────────────────────────────┐
│ DNS Records - bhcosmetics.ma                    │
├────────┬──────────┬─────────────────────┬───────┤
│ Type   │ Name     │ Value               │ TTL   │
├────────┼──────────┼─────────────────────┼───────┤
│ A      │ @        │ 76.76.21.21         │ 3600  │
│ CNAME  │ www      │ cname.vercel-dns.com│ 3600  │
└────────┴──────────┴─────────────────────┴───────┘
```

## الخطوة 4: تفعيل SSL (HTTPS مجاني)

### في Vercel:
1. اذهب إلى **Settings** → **Domains**
2. SSL سيتم تفعيله تلقائياً خلال دقائق
3. سيظهر 🔒 بجانب النطاق
